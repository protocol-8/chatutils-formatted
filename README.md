# ChatUtils

### What it does
It's most significant features include (but are not limited to):
* Increased the maximum amount of chat messages from 100 to 16,384
* Compact Chat
* Animated Chat
* Chat Heads
* Timestamps
* Transparent Chat
* Don't clear the Chat History on disconnect
